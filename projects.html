Home Security System

This project involves the development of a smart home safety device using an ESP32 microcontroller kit. The device is equipped with a PIR (Passive Infrared) motion sensor to detect movement within a designated area. Upon detecting motion, the ESP32 triggers an alert system, which includes a loud buzzer and LED indicators.
